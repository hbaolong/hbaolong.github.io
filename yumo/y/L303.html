博通经籍

bó tōng jīng jí

解释::博：广博：籍：书籍。广博而又精通经典文献。形容人学识渊博。

出处::《后汉书·马融传》：“初，京兆挚恂以儒术教授隐于南山，不应征聘，名重关西。融从其游学，博通经籍，恂奇融才，以女妻之。”

近义词::博古通今

用法::动宾式；作谓语；含褒义

博采众议

bó cǎi zhòng yì

解释::博：广；议：建议。广泛采纳群众的建议

出处::晋·陈寿《三国志·吴志·孙登传》：“诚宜与将相大臣详择时宜，博采众议，宽刑轻赋，均息力役，以顺民望。”

近义词::博采群议

反义词::独断专行

用法::动宾式；作谓语；含褒义

英文::collect widely all advices

博采众长

bó cǎi zhòng cháng

解释::从多方面吸取各家的长处。

示例::他对学生、画友，从不以我画线，而是主张～，自立门户。（黄胄《怀念望云老师》）

近义词::集思广益

反义词::刚愎自用、自以为是

用法::动宾式；作谓语、定语；形容吸收别人的长处

英文::collect widely all good qualities

博闻多识

bó wén duō shí

解释::博：广博；闻：见闻；识：学识。知识丰富，见闻广博

出处::《魏书·李业兴传》：“博闻多识，万门千户，所宜访询。”

示例::明·冯梦龙《东周列国志》第35回：“公子生长中原，～，必知此兽之名。”

近义词::博学多闻、见多识广

反义词::孤陋寡闻

用法::联合式；谓语、定语；含褒义

英文::well learned and informed

博闻强志

bó wén qiáng zhì

解释::博：广博；闻：传闻；志：记住，记忆。指见闻广博，记忆力强

出处::先秦·荀况《荀子·解蔽》：“博闻强志，不合王制。”

示例::西汉·司马迁《史记·屈原贾生列传》：“～，明于治乱。”

近义词::博闻强识

用法::联合式；作谓语、定语；含褒义

英文::have wide learning and a retentive memory

博闻强记

bó wén qiáng jì

解释::形容知识丰富，记忆力强。

出处::《史记·屈原贾生列传》：“博闻强志，明于治乱，娴于辞令。”

示例::这本书的作者一定是位～的学者。

近义词::博古通今、博学多才、博览群书、博文强识

反义词::不学无术、孤陋寡闻

用法::联合式；作谓语、定语；用于称颂人

英文::have wide learning and a retentive memory

博闻辩言

bó wén biàn yán

解释::博：多。闻：传闻。辩言：巧言。形容道听途说，似是而非的言论。

出处::《吕氏春秋·疑似》：“患人之博闻辩言而似通者。”

卜夜卜昼

bǔ yè bǔ zhòu

解释::卜：占卜，选择。形容夜以继日地宴乐无度，没有节制

出处::先秦·左丘明《左传·庄公二十二年》：“臣卜其昼，未卜其夜，不敢。”

示例::清·纪昀《阅微草堂笔记·姑妄听之三》：“又冶荡殊常，蛊惑万状，～，靡有已时。”

近义词::卜昼卜夜

用法::联合式；作谓语、定语；含贬义

英文::day and night

卜昼卜夜

bǔ zhòu bǔ yè

解释::形容夜以继日地宴乐无度。

出处::《左传·庄公二十二年》：“臣卜其昼，未卜其夜，不敢。”

示例::把这些指挥官们，招集在这列花车里，～、昏天黑地，一起比大欢喜。（徐铸成《何成浚与叶蓬》）

近义词::卜夜卜昼

用法::联合式；作谓语、定语；含贬义
