格律分析

文/韩宝龙

2025年12月25日

王维的《陇头吟》是一首七言古诗，借用乐府旧题，在格律上体现了唐代古体诗的特点，即不拘泥于严格的平仄对仗，更注重气势的流转和意境的营造。下面我们从平仄、用韵、对仗和句式等方面进行分析。

📖 平仄分析

作为一首七言古诗，《陇头吟》的平仄安排较为自由，不像近体诗那样有严格的定式。我们来看开篇几句的平仄情况（以中古音为准，“平”用“—”表示，“仄”用“|”表示，可平可仄用“+”标记）： - 长安少年游侠客：—— — | — | | - 夜上戍楼看太白：| | | — — | | - 陇头明月迥临关：| — — | | — — - 陇上行人夜吹笛：| | — — | — |

从以上分析可以看出，诗句的平仄组合富于变化，并未严格遵守近体诗“对”与“黏”的规则。这种处理使得诗歌的节奏更富于变化，情感表达更为自由奔放，与边塞主题的苍茫辽阔相得益彰。

🎵 用韵方式

本诗的用韵体现了古体诗换韵的特点，且韵脚的转换常与诗意的转折相关联： - 第1、2句：以“客”、“白”押韵（入声陌韵），入声短促有力，起笔突兀，契合少年豪情。 - 第3、4句：换为“关”、“笛”押韵（“关”为平声删韵，“笛”为入声锡韵，为通韵转韵），场景转向边关月夜。 - 第5、6句：再换为“愁”、“流”押韵（平声尤韵），韵脚转为悠长平缓，引出老将愁思。 - 第7、8句：转为“战”、“侯”押韵（去声霰韵、平声尤韵，为转韵），语气斩截，形成今昔荣枯的对比。 - 第9、10句：以“国”、“头”押韵（入声职韵、平声尤韵，为通韵转韵），借苏武典故深化主题。

这种多次转换韵脚的方式，使诗歌的声律随着场景切换和情感起伏而不断变化，避免了单调感，增强了音乐的张力和层次感。

⚖️ 对仗情况

古体诗不要求通篇对仗，但诗中常穿插工整或宽泛的对仗以增色。此诗中对仗运用巧妙： - “身经大小百余战，麾下偏裨万户侯”：此联对仗工整，“身经”对“麾下”，“百余战”对“万户侯”，通过鲜明对比突出了老将功高不赏的悲愤。 - “苏武才为典属国，节旄空尽海西头”：以典故暗含对比，可视为意对，深化了功大赏薄的古今同慨之叹。

🔤 句式结构

诗句结构丰富多变，避免了呆板： - 主谓结构：如“陇头明月迥临关”。 - 连动句式：如“夜上戍楼看太白”。 - 活用词语：如“看太白”的“看”字，既写动作，又透露出少年对成功的渴望，简洁传神。

💎 总结

《陇头吟》在格律上充分展现了唐代七言古体诗自由跌宕的特点。诗人通过平仄的灵活交错、韵脚的巧妙转换、局部的精妙对仗以及多样的句式结构，将形式与内容紧密结合，声情并茂地表达了边塞生活的艰辛和将士心中的不平之气。
