First name = 名字 / 教名 / 个人名（中文里最常见的说法就是“名字”）

超清楚对比（2025日常英语）

词 意思 中文叫法 例子
first name 名字（排在前面那个） 名字 / 名 My first name is Emma.
given name 正式说法（和 first name 同义） 名 Given name: Michael
last name / family name / surname 姓 姓 Last name: Jordan
full name 全名 全名 Full name: Michael Jordan
middle name 中间名（很多欧美人有） 中间名 Michael Jeffrey Jordan
nickname 昵称 昵称 MJ, LeBron → “King James”

2025年最常见的问法 - What’s your first name? → 你叫什么名字？
- First name or given name on the form, please. → 请填“名”
- Can I call you by your first name? → 我可以直呼你名字吗？

文化小贴士 - 英语国家：先写/说 first name，再写 last name（Emma Smith）
- 中国：先写姓再写名（王芳 → Fang Wang）

一句话记住：
First name = 你朋友叫你的时候用的那个名字（不是“王先生”“李小姐”里的“王”“李”）。

那么，什么有会有middle name呢？

谁会拥有 middle name？（2025最新规律总结）

国家/文化 拥有 middle name 的比例 典型情况
美国 80–90% 几乎人人有！最爱 middle name 的国家
英国 50–70% 常见，但没美国那么普遍
加拿大 / 澳大利亚 / 新西兰 70–90% 跟美国差不多
爱尔兰 / 苏格兰 60–80% 常把祖先名字或母亲娘家姓当 middle name
西班牙语国家（墨西哥、西班牙、阿根廷等） 90%+ 通常有两个姓+一个或多个名，但“中间的那个名”也算 middle name
法国 20–40% 传统上多个教名，但现在很多人只用一个
德国 / 荷兰 / 北欧 30–60% 常见，但越来越多人不写
东欧（俄罗斯、波兰等） 10–30% 几乎没有，顶多一个父名（patronymic）
中国 / 日本 / 韩国 基本为 0% 完全没有 middle name 概念
印度 5–20% 只有受西方影响的家庭或基督徒才会有
阿拉伯国家 极少 通常是“名字 + 父亲名 + 祖父名 + 部落/家族名”，不算传统 middle name

2025年最常见的 middle name 来源（美国为例）

  1. 家族传承（爷爷奶奶的名字）
  2. 母亲的娘家姓（超火！比如 Elon Musk 的孩子）
  3. 宗教名字（Marie, Ann, Grace, James, John）
  4. 父母最喜欢的随便一个名字（North West, Blue Ivy）
  5. 双中间名也越来越流行（Emma Rose Marie Smith）

一句话总结（背下来就行）

基本上只有“英语国家+西班牙语国家”的人才会系统性地拥有 middle name，其中美国人最痴迷，几乎人手一个。

你认识的外国人如果有 middle name，99%来自上面这些地方！
