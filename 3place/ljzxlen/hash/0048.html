‘三元一次方程’，你买几次的？

文/韩宝龙

hbaolong@vip.qq.com

群内原文

3.3 Only a proposition has sense; only in the context of a proposition does a name have meaning.

命题才有意义；

一个名字，只有在命题中，才有意义。

这两句，很有意思。

有一次，一个朋友问我：道是什么？

我说，哪里的‘道’？

她说：就是哪个道呀？

我说：哦，就是路的意思。

其实，’道’这个名词，没有任何意义，只有把它放在一个命题里才有意义。

这也就同样表达了：

只有命题才有意义。

当我说“泡泡是一个美女”的时候，我表达的就是一个意义：我喜欢泡泡。

而简单的词：泡泡，美女，并没有任何意义。

当然，当我说：泡泡！

这句话其实是：“泡泡！你在哪里？听到了回答我一声”，这是完整的命题。

3.31 Every part of a proposition that characterizes its sense I call an expression (a symbol).

命题的每一个部分都在表达一定的意义，这个意义就我定义为公式（符号）。

(The proposition itself is an expression.)

命题本身也是一个公式。

我喜欢泡泡。这是一个命题，

也是一个公式，或者表达式：

X喜欢Y。

我喜欢泡泡的绿色裙子，这是一个大命题：

命题1：泡泡穿着绿色裙子；

命题2：我喜欢穿着绿色裙子的泡泡。

这两个明天就是两个公式，而整个命题也是一个公式，更复杂的公式。

Expression is everything essential to the sense of a proposition that propositions can have in common with one another.

公式，是命题的一切。而也是所有的命题之间共有的。

比如，我喜欢泡泡，我喜欢龙儿，这是两个命题，他们的公式都是：X喜欢Y，这个公式（表达式）是这两个命题共有的。

An expression marks a form and a content.

一个表达式，标记了形式和内容。

X喜欢Y，就是形式，和内容。

3.311 An expression presupposes the forms of all propositions in which it can occur. It is the common characteristic mark of a class of propositions.

一个公式（表达式）预设了所有命题的形式（X喜欢Y）所可能出现的情况。

比如，我喜欢狗，狗喜欢泡泡，所有的可能性都存在。

对于一系列命题来说，他们有相同的表达结构。

3.312 It is thus represented by the general form of the propositions that it characterizes.

 That is to say, in this form the expression will be constant and everything else variable.

这样一来，公式就是由命题的一般形式来表示的。

也就是说，在公式（表达式）的形式里，有常量和除了常量以外的变量。

X喜欢Y，喜欢是常量，X和Y都是变量，他们可以是很多可能。

比如，我喜欢泡泡，我喜欢龙儿，我喜欢狗，狗喜欢泡泡。

说到这里，咱们知道了：

其实，就是一个有函数，多元多次函数。有常量，有变量。而函数的结构是不变的。

这个很好理解吧。

比如

f(x,y,z)=x+y+8z

3.313 An expression is thus represented by a variable whose values are the propositions that contain the expression.

所以，一个公式，就是用变量表达，那些包含在命题表达式里的值。

比如，表白（X，Y）=X 喜欢 Y

这个公式就是被变量XY在命题中的值来表达的。

(In the limiting case the variable becomes a constant, the expression a proposition.)

这些变量的取值范围，肯定不是无限的，而是有限制的。

变量在限制范围内变成常量。

比如，我喜欢泡泡。X变成了常量‘我’，Y变成了常量‘泡泡’。这个公式也就变成了命题。

I call such a variable a ‘propositional variable’.这个可变性，就是‘命题的可变性’。

3.314 An expression only has meaning in a proposition. Every variable can be construed as a propositional variable.

一个公式（表达式）只能拥有有意义的命题，每一个可变的参数，都会被分解为命题参数。

这个很好理解。

3.315 If we change a constituent of a proposition into a variable, then there is a class of propositions that are all the values of the resulting variable proposition. This class still depends in general on what we mean, according to our arbitrary conventions, by parts of the original proposition. However, if we change into variables all those signs whose meaning was arbitrarily determined, then there is still such a class. But this is not now dependent on any convention, only on the nature of the proposition. It corresponds to a logical form—a logical prototype.

如果我们把一个命题的一部分改成变量，那么就会出现一系列的命题。这个系列命题就有了所有的值作为结果的命题。

比如，命题：我喜欢泡泡，我把‘喜欢’变成变量，那么，就有了很多可能性：

我讨厌泡泡，我爱泡泡，我需要泡泡，我恨泡泡等等。

这一些系列命题，依然会受限于我们的生活习俗。

比如，我不能说：我狗泡泡。

同时，如果，我们把所有的常量，都变成变量，这依然会形成一个新的类。

X F Y

这就不依赖任何形式了，只是一个纯纯的逻辑原型了。

也就没有了除逻辑以外的任何意义了。

3.316 What values a propositional variable may take is fixed.

明天的变量的值，其实都是固定的。

比如，我X你之间，也就那么几个词。

The fixing of the values is the variable.

3.317 The fixing of the values of propositional variables is the specification of the propositions whose common mark is the variable.

明天变量的那些固定的值，是对命题的共同标记的规定。

比如，X的体重是Y，这个X、Y就是两个变量。他们的取值范围，就是对这个表达式的规定。

比如，你不能说：泡泡体重1000公斤。

The fixing is a description of these propositions.

固定值，或者说，取值范围，就是对这些命题的描述。

The fixing will thus be a matter only of symbols, not of their meaning.

所以，取值范围，只是符号问题，不是意义。意义在形式。

And only this is essential to the fixing, that it is only a description of symbols and states nothing about what is signified.

对于变量的取值范围来说，只是符号、状态的描述意义，而对它要表达的内容没有其他意义。

其他意义，都在公式里，也就是表达式里。

The way in which the propositions are described is inessential.

描述命题的方式是不重要的，重要的是明天的公式。

3.318 I construe a proposition—like Frege and Russell—as a function of the expressions contained in it.

我们可以把命题分解为一个包括了内容的表达式。

比如：我喜欢泡泡。就是

喜欢（x，y）=X喜欢Y

今天的内容，主要是：

命题=公式+变量，或者

f(x,y,z)= 3x+4y+5Z

以上。

《逻辑哲学论》3.3–3.318 完整讲义

——“命题＝函数，名字＝参数，意义＝形式”

它是一个函数，
名字是参数，
意义是函数形式，
变量填死＝命题亮灯。
