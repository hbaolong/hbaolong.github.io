人物小传：周邦彦

文/韩宝龙

2026年2月19日 17时34分14秒

人物小传

周邦彦是北宋末年最杰出的词人之一，被尊为婉约词的集大成者和格律词派的创始人。他在中国词史上的地位极高，旧时词论称他为 “词家之冠” 或 “词中老杜”（意为词中的杜甫），是公认“负一代词名”的词人 。他上承柳永、秦观，下开南宋姜夔、吴文英一派，完成了婉约词在艺术形式上的高峰 。

📜 周邦彦：核心档案

词集 《片玉集》（又名《清真集》）

🏛️ 生平：从献赋得官到执掌大晟府

周邦彦的一生与北宋后期的政治风云紧密相连。

  • 早年与献赋得官：他年少时“疏隽少检”（性格放浪），但极为勤学 。宋神宗元丰年间，他以布衣身份入京为太学生。元丰六年（1083年），他献上长达七千余字的《汴都赋》，歌颂新法和新政，神宗览后大加赞赏，破格提拔他为太学正（太学学官），由此一举成名 。
  • 外放州县：神宗去世后，旧党执政，周邦彦因属于新党一派，被排挤出京，在庐州（今安徽合肥）、荆州、溧水等地担任地方官近十年，这段“浮沉州县”的经历让他的词作更加深沉 。
  • 重返京师：哲宗亲政后，重行新法，因再次进献《汴都赋》而得到召回，任国子监主簿、校书郎等职 。
  • 提举大晟府：徽宗时期，他因精通音律，被任命为提举大晟府（国家最高音乐机构），负责谱制词曲，供奉朝廷。他利用这个职务，审订古调、创制新曲，将词体的格律推向了极致 。
  • 晚年与逝世：晚年他又外调顺昌府、处州等地，后定居南京（今河南商丘），宣和三年（1121年）病逝，享年66岁 。

✨ 词史地位与艺术成就

周邦彦在词史上的地位极高，他的创作标志着北宋婉约词的成熟与定型。

  1. 集大成者，承前启后 他继承柳永的慢词铺叙、秦观的婉约情致、苏轼的词境开拓，并将这些融为一体，形成自己独特的风格。唐圭璋先生评价他：“北宋婉约作家，周最晚出，熏沐往哲，涵泳时贤，集其大成” 。同时，他开南宋姜夔、吴文英、史达祖等格律词派之先河 。

  2. 精通音律，创调之才 他是音乐家，能自度新曲，《六丑》《兰陵王》《瑞龙吟》等词调皆为其所创 。他讲究四声清浊，格律极严，使词与音乐达到了高度和谐，被后世词人奉为创作圭臬。王国维说他“创调之才多，创意之才少”，承认他在音律形式上的巨大贡献 。

  3. 长于铺叙，善用典故 他的长调慢词极善铺陈展开，结构曲折回环，章法严密 。同时，他善于融化前人诗句入词，将唐诗意境巧妙融入词中，自然天成，所谓“多用唐人诗语，檃括入律，浑然天成” 。

  4. 语言典丽，意境浑成 他的语言精工典雅，却不失自然。如“叶上初阳干宿雨、水面清圆，一一风荷举”（《苏幕遮》）用最简练的文字画出荷塘神韵。又如“烟中列岫青无数，雁背夕阳红欲暮”（《玉楼春》），意象开阔而含蓄。

🎭 轶事典故：与李师师的绯闻

周邦彦在民间最广为人知的，是他与名妓李师师、宋徽宗之间的“三角恋”传说 。

  • 《少年游》的由来：据宋人笔记记载，周邦彦正与李师师幽会时，宋徽宗突然驾到，周邦彦情急之下躲到床下。徽宗带来江南新贡的橙子，与李师师调情。周邦彦事后将所见所闻写成 《少年游·并刀如水》：“并刀如水，吴盐胜雪，纤指破新橙。锦幄初温，兽香不断，相对坐调笙……” 。
  • 因词得祸，又因词得福：后来李师师当着徽宗的面唱了这首词，徽宗大怒，将周邦彦贬出京城。师师送别时，周邦彦又写下 《兰陵王·柳》，其中有“愁一箭风快，半篙波暖，回头迢递便数驿，望人在天北”之句。徽宗听了此词后，觉得周邦彦确实是人才，又将他召回，任命为大晟府提举 。

注：这段逸事多见于宋人笔记（如《贵耳集》《耆旧续闻》），正史无载，真实性存疑，但广为流传，成为文学史上的著名掌故 。

📝 历史评价

  • 南宋陈振孙：“清真词多用唐人诗语，檃括入律，浑然天成；长调尤善铺叙，富艳精工，词人之甲乙也。”
  • 清戈载：“清真之词，其意淡远，其气浑厚，其音节又复精妍和雅，最为词家之正宗。”
  • 王国维（《人间词话》）：“美成深远之致，不及欧、秦，唯言情体物，穷极工巧，故不失为第一流之作者。” 他晚年又在《清真先生遗事》中赞周邦彦为“词中老杜”，认为其集大成地位无可撼动 。

💎 总结与关联

周邦彦是北宋词坛的最后一座高峰。他生在北宋末年，国势衰微而文风犹盛，他以艺术家的匠心，将词的形式之美推到了极致。

与你之前关注的词人相比： * 李煜是以血写词的亡国之君，直抒胸臆，天然绝妙。 * 秦观是“古之伤心人”，以清丽柔婉写悲情。 * 晏几道是“古之伤心人”，以真挚深婉写回忆。 * 周邦彦则是以匠心作词的格律大师，他更重技巧、音律和章法，是文人词成熟的标志。
