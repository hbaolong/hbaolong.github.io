拜占庭帝国（Byzantine Empire）
官方自称：罗马帝国（Βασιλεία τῶν Ῥωμαίων / Imperium Romanum）
我们今天叫它“拜占庭”，是为了方便区分“西罗马”和“东罗马”，但他们自己从头到尾都认为：
“我就是罗马帝国，只是首都搬到君士坦丁堡了，别的都没变。”

一句话总结

拜占庭帝国 = 活了1123年的东罗马帝国（330–1453年），
它是古罗马文明在希腊化＋基督教包装下的“续命1123年豪华版”，
也是中世纪欧洲、近东、伊斯兰世界、斯拉夫世界三者之间的超级缓冲国、文明灯塔和金钱印钞机。

最硬核数据（2025年视角回顾）

项目 内容
正式起止时间 330年君士坦丁大帝迁都拜占庭（改名君士坦丁堡）
→ 1453年5月29日君士坦丁堡被奥斯曼土耳其攻陷
最长寿帝国 连续存在1123年（西罗马476年就没了）
巅峰疆域（565年查士丁尼一世） 西起西班牙直布罗陀，东到美索不达米亚，南到埃及，北到多瑙河，地中海彻底变“罗马湖”
首都 君士坦丁堡（今天的伊斯坦布尔），千年不换
官方语言 前期拉丁语 → 7世纪后彻底希腊语
宗教 东正教（1054年东西教会大分裂后正式独立）
货币 金币“索利都斯/贝占特”（Solidus/Bezant），中世纪最硬的国际硬通货，稳定700年
终极死因 1453年奥斯曼苏丹穆罕默德二世用巨炮轰塌城墙，末代皇帝君士坦丁十一世战死城头

拜占庭帝国的“四大标签”

标签 代表性画面
1. 罗马的正统继承者 皇帝自称“罗马人的皇帝”（Basileus ton Rhomaion），法律用《查士丁尼法典》
2. 希腊化的基督教帝国 语言希腊语、宗教东正教、艺术镶嵌画＋穹顶（圣索菲亚大教堂）
3. 中世纪最富、最文明的城市 君士坦丁堡
4. 千年打不死的小强 被阿拉伯、波斯、保加尔、维京、突厥、十字军轮番打，却活到1453年

历史阶段简表（好记版）

时期 时间 关键词
早期拜占庭 330–717 查士丁尼、收复西地中海、瘟疫、阿拉伯崛起
马其顿复兴 867–1056 巴西尔二世“保加利亚屠夫”、拜占庭黄金时代
科穆宁王朝 1081–1185 阿莱克修斯一世求援→第一次十字军
第四次十字军劫难 1204 威尼斯＋十字军洗劫君士坦丁堡，帝国分裂
拉丁帝国＋尼西亚流亡 1204–1261 拜占庭人跑去小亚细亚建临时政府
帕里奥洛格复兴 1261–1453 收复君士坦丁堡，但已病入膏肓
终结 1453 奥斯曼巨炮轰城，末帝战死

它死后留给世界的遗产（今天还在用）

  1. 东正教（俄罗斯、希腊、塞尔维亚、罗马尼亚等1500万信徒的源头）
  2. 《查士丁尼法典》→ 欧洲大陆法系的真正鼻祖
  3. 希腊火（古代“凝固汽油弹”）技术失传，但吓得阿拉伯人几百年不敢打海战
  4. 君士坦丁堡的学术传统 → 1453年学者带古希腊手稿逃到意大利 → 文艺复兴直接点火

一句话最直观的总结

拜占庭帝国就是：
穿着罗马盔甲、说着希腊语、信着东正教、住在君士坦丁堡、用金币统治地中海、活了整整一千多年的“罗马僵尸”。
它死的那一天（1453年5月29日），正好被视为中世纪的终结、近代的开端。

所以你现在去伊斯坦布尔看圣索菲亚大教堂，其实就是在看：
一个从罗马共和国一路活到奥斯曼帝国的超级老妖怪留下的遗体。
