马丁·路德（Martin Luther，1483–1546）
宗教改革之父、基督教历史上最猛的“钉子户”、把欧洲劈成天主教和新教两半的德国修士。

一句话标签：
那个1517年10月31日在维滕贝格教堂大门上“咚咚咚”钉了95条论纲，然后把整个中世纪教会秩序砸得稀巴烂的狂人。

2025年硬核数据（一眼记住）

项目 内容
出生 1483.11.10　德国图林根艾斯莱本（矿工家庭）
身份转变 奥古斯丁会修士 → 神学博士 → 被绝罚的“异端” → 新教之父
最著名事件 1517.10.31　钉《九十五条论纲》
核心口号 唯独圣经（Sola Scriptura）
唯独恩典（Sola Gratia）
唯独信心（Sola Fide）
最硬核行为 1521年沃尔姆斯帝国会议当面拒绝向皇帝和教宗认错：“除非用圣经说服我，否则我绝不收回！”
结果 1521年被教宗绝罚 + 被帝国通缉 → 萨克森选帝侯腓特烈三世把他藏起来

死因 1546.2.18　中风去世，享年62岁
2025年遗产 全球新教徒约9–10亿（路德宗、加尔文宗、浸信会、卫理公会等都算他的后代）

他到底干了哪几件把欧洲炸上天的事？

  1. 95条论纲（1517）
    直接开怼“赎罪券”：你拿钱买赎罪券等于买空气，救不了灵魂。
    → 相当于在中世纪的“微信朋友圈”里公开骂教皇是骗子。

  2. 1521年沃尔姆斯帝国会议
    皇帝查理五世+全体诸侯+教廷代表让他认错。
    他最后一句：“我站都站不稳了，除非用圣经说服我，否则我绝不收回！”
    → 当场被宣布为“异端+帝国敌人”，赏金通缉。

  3. 翻译德文圣经（1522–1534）
    把拉丁文《武加大圣经》翻译成平民德语，让农民都能自己读经。
    → 相当于把“教会垄断解释权”砸烂，知识民主化的核弹。

  4. 娶了修女（1525）
    和逃出修道院的加塔琳娜·冯·博拉结婚，公开说“神父可以结婚”。
    → 把独身制传统直接踹翻。

一句话最狠总结

马丁·路德就是那个：
用一把锤子、一张纸、一本德文圣经，把中世纪教会千年铁板钉出了95个洞，
然后顺手把基督教世界劈成两半，
一半叫天主教，一半叫新教，
这一劈，就是500年，到现在还没完全缝上。

他死后30年，欧洲直接打起了“三十年战争”（1618–1648），死了800万人。
所以你今天看到的新教（路德宗、长老会、浸信会……），
全是这个德国修士500年前一锤子砸出来的。
