1. 这句话的正确历史版本应该是：

9–10世纪，丹麦维京人（以及挪威维京人）多次摧毁、洗劫、占领了不列颠群岛上的盎格鲁-撒克逊诸王国，史称“大异教徒军队”（Great Heathen Army）时期。

关键时间点： - 865年：传说中的“洛瑟尔（Ragnar Lodbrok）的儿子们”率领大规模丹麦舰队登陆英格兰东海岸。 - 866–878年：先后灭亡了诺森布里亚、麦西亚、东盎格利亚三个盎格鲁-撒克逊王国。 - 878年：只有威塞克斯王国（阿尔弗雷德大帝）在埃丁顿战役险胜，才挡住了丹麦人统一全英格兰的步伐。 - 后来丹麦国王“八字胡斯韦恩”和“血斧埃里克”、克努特大帝（1016–1035）真的征服了整个英格兰。

所以“9世纪丹麦摧毁蛮族”这句话，如果把“蛮族”理解成“当时盎格鲁-撒克逊人眼里的异教维京蛮族”，那它就是真的；如果把“蛮族”当成某个具体部落，就又变成谣言了。

2. 这和《哈姆雷特》到底有没有关系？

有，而且是非常直接的关系，只是被中文网络梗扭曲了。

《哈姆雷特》的故事原型来自12世纪丹麦史学家萨克索（Saxo Grammaticus）的《丹麦史》（Gesta Danorum）。萨克索记载了一个发生在维京时代早期（传说8–9世纪）的丹麦王子复仇故事：

  • 丹麦国王被弟弟冯格（Feng）谋杀
  • 王子阿姆莱特（Amlethus）装疯卖傻
  • 最后血洗王宫复仇，全族死光

莎士比亚直接把这个故事改编成了《哈姆雷特》。

而萨克索写这段故事的时候，用的正是9世纪维京时代的那套血腥复仇文化背景（家族仇杀、装疯、血婚、火葬船……全都有）。

所以：

《哈姆雷特》的故事背景 ≈ 9世纪的丹麦维京社会
而9世纪的丹麦维京人确实把不列颠的盎格鲁-撒克逊人打得七零八落

3. 中文网络梗的最终形成路径大概是：

9世纪丹麦维京人暴打盎格鲁-撒克逊蛮族
→ 哈姆雷特的故事就发生在9世纪丹麦
→ 哈姆雷特最后全家死光，王国乱套
→ 有人开玩笑说“所以丹麦后来被灭国了”
→ 再一传十十传百，就变成了“9世纪丹麦摧毁蛮族”“哈姆雷特复仇失败导致丹麦亡国”……
→ 最后甚至传成了“丹麦把满族灭了”这种离谱版本

正确答案总结

  • “9世纪丹麦摧毁蛮族”：半真半假。真的部分是丹麦维京人确实把当时的不列颠“蛮族”（盎格鲁-撒克逊人）打得很惨。
  • 和《哈姆雷特》有关系：故事原型就发生在那个时代、那个文化的丹麦。
  • 但哈姆雷特本人和丹麦亡国没有任何关系，那纯粹是中文互联网的“历史恐怖故事”式演绎。

一句话概括：
《哈姆雷特》是9世纪维京丹麦的“复仇王子”故事”，而9世纪丹麦维京人确实把英国的蛮族揍得满地找牙，两件事同框没毛病，但把它们因果串起来就是段子了。

什么是维京人？

维京人（Vikings）
不是一个民族，而是一份“职业＋生活方式”的总称。
简单说：8–11世纪北欧（丹麦、挪威、瑞典）那些出海抢劫、贸易、殖民、探险的海上战士和商人，就叫维京人。

最直白的定义（三句话记住）

  1. 时间：约793年（林迪斯法恩修道院被袭）到1066年（黑斯廷斯战役威廉征服英格兰）
    → 这273年叫“维京时代”（Viking Age）。

  2. 地域：主要来自今天的三国

    • 丹麦维京（Danes）→ 去英格兰、法国、爱尔兰
    • 挪威维京（N）→ 去冰岛、格陵兰、北美、苏格兰、诺曼底
    • 瑞典维京（S）→ 往东走，去俄罗斯、乌克兰、君士坦丁堡（他们自称“瓦良格人”Varangians）

  3. 核心身份：
    “Viking”这个词本身在古诺尔斯语里是动名词，意思是“出海远征/抢劫”。
    所以北欧人平时是农民、渔民、商人，只有“出去干维京”的时候才叫Vikingr（维京人）。

    → 就像今天说“他在跑外卖”“他在当海盗”，不是固定身份，而是一种行为。

维京人干了什么？（四大业务版图）

方向 代表事件与成果
西征 793年洗劫林迪斯法恩 → 865年“大异教徒军队”占领英格兰大半 → 克努特大帝1035年统一英格兰、丹麦、挪威

维京人最后去哪儿了？

1066年黑斯廷斯战役后，“维京时代”正式结束。
他们没有消失，而是彻底融入欧洲：

  • 英格兰、爱尔兰 → 变成了英国贵族（诺曼人）
  • 法国 → 变成了诺曼底公爵（后征服英格兰）
  • 冰岛、格陵兰 → 后裔还在
  • 俄罗斯、乌克兰 → 变成了基辅罗斯贵族
  • 君士坦丁堡 → 当了几百年的瓦良格卫队

一句话最直观的总结

维京人不是一个“民族”，而是8–11世纪北欧人的一次“集体创业潮”：
一群超级能打的海盗、商人、探险家，拿着当时世界最先进的龙船，
把整个欧洲从西到东都跑了一遍，抢完钱、占地盘、娶老婆、建国家，
最后摇身一变成了英国人、法国人、俄罗斯人、冰岛人……
把自己的基因和传说留在了半个地球。

所以今天你看到金发碧眼的冰岛人、英国王室的诺曼姓氏、俄罗斯国名的“罗斯”来源，
背后都站着一群一千年前喊着“Skoal！”干杯的维京老祖宗。
