词牌介绍：玉楼春

文/韩宝龙

2026年3月18日 11时22分37秒

《玉楼春》是一个形似七言律诗、实为仄韵词调的经典词牌。它以整齐的七言八句、密集的仄声韵脚，营造出沉重压抑、适合抒发惆怅感怀的独特声情，在宋词中与《木兰花》长期相混，成为词学史上一对著名的“双胞胎”。

🔤 名称渊源：从“玉楼春色”到词牌名

“玉楼春”之名，直接取自晚唐五代的词句：

其一，取自顾夐词。 五代后蜀词人顾夐《玉楼春》词有“月照玉楼春漏促”和“柳映玉楼春欲晚”之句，后人取其中“玉楼春”三字为调名 。

其二，取自欧阳炯词。 《尊前集》中欧阳炯词也有“日照玉楼花似锦”和“春早玉楼烟雨夜”句，同样是调名的来源 。

其三，别名之繁。 李煜词名“惜春容”，朱敦儒词名“西湖曲”，康与之词名“玉楼春令”，《高丽史·乐志》名“归朝欢令” 。十余个别名，见证了它在宋代的广泛流传。

格律核心密码 ：

  1. 形似七律，实为词调：此调由八句七言组成，看似与七言律诗无异，但通篇押仄韵（七律押平韵），且六个句子用韵（只有第三、七句不押韵），格律与诗体迥异。

  2. 仄韵密集，声情沉重：全词共押六个仄声韵，韵脚密集，形成一种沉重压抑、呜咽顿挫的声情，最适合表达沉闷、惆怅、感怀之情 。

  3. 四句相同句式：全词八句中，有六个句子是“中仄中平平仄仄”句式，占了绝对主导地位，形成反复回环的韵律效果。

  4. 对仗灵活：前段第三、四句和后段第三、四句可以对偶，如晏殊“旋开杨柳绿蛾眉，暗拆海棠红粉面”，工致可观 。

💡 声情灵魂与创作要诀

此调之魂，在于其整齐的七言句式与密集的仄声韵脚之间形成的巨大张力——看似工整如律诗，实则呜咽如泣诉。

一、声情特征：沉重压抑，惆怅感怀

  • 适合题材：此调因仄声韵较密，且有四个仄起律句，因而声情较为沉重压抑，适于表达沉闷、惆怅、感怀之情 。
  • 不宜豪放：不适合抒发慷慨激昂的情感，更宜婉约、惋惜、怀旧、抒情 。
  • 名家用法：欧阳修用此调作词二十余首，多写花间尊前情事，亦有酬赠、写景、咏物、离情等作 。辛弃疾词十七首，则多用以赠酬，甚至有为词论之戏作 。

二、创作核心技法

  1. 上下片结尾宜对仗：上下片最后两个七字句（即第三、四句），一般要对仗处理，形成工致之美。

  2. 韵脚密集须起伏：全词六仄韵，须在密集的押韵中形成轻重层次，避免单调熟滑。

  3. 句法稳定，适合初学：此调的写作要求较宽，句法稳定，是最适合初学者练习的词牌之一。

  4. 注意与《木兰花》的混淆：宋代因《尊前集》误刻，《玉楼春》与《木兰花》被混为一谈，实则二者音谱不同 。李煜此词在《尊前集》中即被误刻为《木兰花》，宋人遂将错就错 。

🏮 名篇赏析：欧阳修《玉楼春·尊前拟把归期说》

宋景祐元年（1034年），欧阳修在西京洛阳的任期已满，要离开居住了四年的洛阳。这首词作于离筵之上，被誉为欧词“于豪放之中有沉着之致”的典范 。

尊前拟把归期说，欲语春容先惨咽。 人生自是有情痴，此恨不关风与月。 离歌且莫翻新阕，一曲能教肠寸结。 直须看尽洛城花，始共春风容易别。

上片：尊前本应欢乐，却要说归期，未语而美人先泣。“拟把”与“欲语”两个层次的递进，写出多少不忍念及和不忍道出的宛转深情 。后两句陡然从眼前情事转入人生哲理——人之情痴发自本性，与风月无关，将个人离愁推广到对整个人世的认知 。

下片：“离歌且莫翻新阕”写离歌已使人肠断，何必再唱新曲？末两句却突然扬起——要“看尽洛城花”再与春风告别，于沉痛中生出遣玩的豪兴。王国维评曰：“于豪放之中有沉着之致，所以尤高” 。

叶嘉莹先生精辟指出：这首词正是欧词风格的最大特色——在赏爱之深情与沉重之悲慨两种情绪相摩荡之中，产生出想要以遣玩之意兴挣脱沉痛之悲慨的一种既豪宕又沉着的力量 。

掌握《玉楼春》的关键：理解其形似七律、实为仄韵词调的本质。此调之难，不在格律之繁（它是初学者最易入手的词牌之一），而在如何在整齐的七言句式中，通过仄韵的密集回环，写出“于豪放之中有沉着之致”的情感张力。正如欧阳修“人生自是有情痴，此恨不关风与月”——十四字写尽千古离愁，又从中见出人生哲理，方为《玉楼春》之正声。
