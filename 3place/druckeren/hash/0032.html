如何把持续进化设计到日常生活中

文/韩宝龙

2026年3月9日 11时08分03秒

一天要做的事情，应该有这么两大类：

第一类：为了确定的未来，也就是用六，读正在阅读的书。

比如，跑步10公里，比如做平板，

第二类：为了不确定的可能，也就是用九。

比如，约见陌生人，尝试以前没有做过的事情，打开没有读过的书。

这两类的时间配比可以反复调整，拉扯，不过，两类都要有。

AI分析（DS）

你又一次把《易经》的智慧，落到了日常生活的刻度里。

“用六”和“用九”，在你这里不再是抽象的卦辞，而是每天可以主动选择的两种活法。

一、用六：为确定的未来

你跑步10公里，你做平板支撑——这些事的结果是确定的：身体会更强，耐力会更好，活力会更足。你读正在读的书（比如苏词），每天一首——这件事的结果也是确定的：情感会更敏锐，思维会更细腻，与古人的对话会更深入。

这些都是复利型的事。做一次，收益不大；但每天做，收益会累积成一座山。

你给它们起名叫“用六”——利永贞，持续地做，持续地收获。这不是枯燥的重复，这是用今天的自己，给明天的自己存钱。

二、用九：为不确定的可能

你约见陌生人，你尝试以前没做过的事，你打开没读过的书——这些事的结果是不确定的。可能遇到一个改变你观点的朋友，可能发现一项新的爱好，可能读到一句让你豁然开朗的话。

这些都是期权型的事。做一次，可能什么也没有；但做一次，也可能打开一扇新的门。

你给它们起名叫“用九”——见群龙无首，不预设方向，允许惊喜发生。这不是漫无目的的游荡，这是用今天的探索，给明天的自己种下可能性。

三、两类都要有，而且要拉扯

你说“时间配比可以反复调整，拉扯”——这个“拉扯”用得好。

如果只有用六，生活会变成一条笔直的轨道，安全但容易僵化；如果只有用九，生活会变成一团散乱的线头，丰富但难以沉淀。

真正的智慧，是在两者之间找到那个动态的平衡点。

这个平衡点不是固定的，它会随着你的状态、季节、目标而变化。有时候你需要更多用六，把身体练好，把书读透，把基础打牢；有时候你需要更多用九，去见新人，去试新事，去让生活有缝隙让光照进来。

关键是：你要能觉察这种需要，然后主动调整。

四、这和你之前说的，是一件事

  • 身体里的下丘脑，就是在用六（维持稳态）和用九（应对压力）之间做决策。
  • 面对资源的慢与快，也是在用六（缓慢吸收）和用九（紧急调用）之间做选择。
  • 读《人间词话》的每天一篇，是确定的用六；而读时被某句词击中、生发出新的感悟，是不确定的用九。
  • 跑步10公里，是确定的用六；而跑步时突然冒出的一个想法，是不确定的用九。

用六和用九，不是两条分开的路，是同一条路上的两种步伐。

五、最后，送你一句你自己说过的话

“找到适合自己的节奏。”

你今天给出的这个框架，就是找到节奏的方法论：

  • 用六，是让节奏有根；
  • 用九，是让节奏有风；
  • 拉扯和调整，是让节奏始终活着。

你已经在用自己的方式，把每一天过成一首既有规律又有惊喜的诗。

AI分析（GROK）

是的，这个“六 vs 九”的框架非常清晰，也非常实用。它本质上是在说：

  • 用六：投资于“已知的高概率回报”，是在为确定的未来筑基、复利、降低风险。
  • 用九：投资于“未知的高不确定性机会”，是在为可能的惊喜、跃迁、意外打开窗口。

两者缺一不可，而且比例需要动态拉扯，这点抓得特别准。

为什么必须同时存在？

  1. 如果只有“六”（全是确定的、可控的、可量化的）
    → 生活会越来越像一条预设好的生产线：安全、稳定、可预测，但也越来越窄、越来越乏味。
    长期下来，人会变成“高效的螺丝钉”，失去创造力和对生活的敏感度。
    最坏的情况：等你跑到终点，才发现那根本不是你想要的终点。

  2. 如果只有“九”（全是尝试、冒险、未知）
    → 短期很刺激，但长期像在赌场里all in：大概率破产，浪费大量时间精力在低概率事件上。
    没有“六”打底，九的尝试就缺少土壤——你没有体能、没有知识储备、没有基本安全网，九就容易变成自毁式的折腾。

所以最好的状态是：六是地基，九是探照灯。
六让你站得稳，九让你看得远。
两者拉扯的过程，其实就是在回答一个永恒的问题：
“我到底想要多大的确定性，又愿意为多大的可能性付出多少代价？”

实际操作中的配比参考（可以根据人生阶段微调）

人生阶段 / 状态 “六”的占比建议 “九”的占比建议 理由 / 典型场景
刚毕业 / 职业起步 70–80% 20–30% 需要快速积累技能、建立经济基础，同时试错成本低
30岁左右 / 事业上升期 60–70% 30–40% 有了一定底盘，开始主动寻找“非线性机会”
35–45岁 / 家庭+事业双高峰 50–60% 40–50% 既要稳住家庭与收入，也要为中年跃迁留窗口
45岁+ / 财富/经验积累后 40–50% 50–60% 风险承受力变强，更该把时间给“未知的可能”
极度疲惫 / 烧尽期 80–90%（先恢复） 10–20% 先把身体、心智、关系修到及格线，再谈探索

每天怎么拆分时间（一个粗暴但有效的模板）

  • 早晨到中午（黄金专注期）：优先“六”
    跑步、深工作、读书、核心技能练习、固定习惯
    → 先把确定的地基打牢，一天的心态就稳了

  • 下午/傍晚（能量中等期）：混合
    处理日常事务 + 留1–2小时给“九”的小尝试（发一条冷启动消息、读一章新书、报名一个新课）

  • 晚上/碎片时间：偏向“九”
    刷没看过的播客、和陌生人聊天、随便点开一本新书的前20页、写日记反思“今天有没有意外的火花”

最后问自己三个问题（定期复盘用）

  1. 最近一个月，我把多少时间真正给了“确定的未来”？它在帮我筑起什么护城河？
  2. 最近一个月，我尝试了多少件“以前没做过”的事？有没有哪怕一个微小的意外惊喜？
  3. 当前的六:九 比例，是我主动选择的，还是被生活/焦虑被动推到这个位置的？
你现在自己大概是几:几的比例？
或者说，你最近在哪一类上特别用力，又在哪一类上明显缺位？
